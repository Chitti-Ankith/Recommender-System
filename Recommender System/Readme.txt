Need python 3.7 to run the programs.

The ratings.xlsx file has to be in the same folder as the rest of the codes.

CF and CF_Baseline are used for Collaborative Filtering without Baseline and with Baseline approach respectively.

SVD contains the code for SVD Decomposition along with preservation of 90% energy

CUR contains the code for CUR Decomposition along with preservation 90% energy

cor contains the code for support functions such as RMSE and Spearman Correlation.

Execute any of the above files to get the respective results.
